# Netflix Data Analysis Project

This project analyzes the Netflix Movies and TV Shows dataset using Python and Pandas.

## Objective

* Explore the dataset
* Perform data cleaning
* Analyze content patterns
* Extract meaningful insights

## Tools Used

* Python
* Pandas

## Dataset

The dataset contains information about Netflix content including type, genre, country, release year, duration, and ratings.

## Key Analysis Performed

* Movies vs TV Shows comparison
* Most popular genres
* Content distribution over years
* Top countries producing content
* Duration analysis of movies
* Ratings distribution

## Key Insights

* Movies are more than TV Shows on Netflix
* Drama and Comedy are among the most popular genres
* USA produces the highest amount of content
* Content production has increased over the years
* The average movie duration is around 99.57 minutes

## Conclusion

This project helped in understanding real-world data analysis including data cleaning, transformation, and extracting insights using Pandas.
